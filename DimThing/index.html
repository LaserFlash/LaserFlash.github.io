Welcome to DimThing.

DimThing allows you to dim your displays if they are too bright

Current version: DimThing 1.0

DimThing Features

  • Varying levels of dimness
  • Multiple Monitors
  • Mode Toggling to include or exclude primary monitor
  • Smooth Transitions
  • Keyboard Shortcuts

Mode Toggle

This allows the dimming ability to be toggled on or off on the primary screen.
Inteneded use for this mode is to consme media such as games or videos on the primary display, and being able to dim any seconday displays seperatly in order to remove distraction due to brightness of other displays.
This does however mean this feature is only avliable on setups with multiple monitors

Keyboard shortcut is ALT + |

Shortcuts

Increase Dimness

  • ALT + ADD
  • ALT + PLUS

Decrease Dimness

  • ALT + SUBSTRACT
  • ALT + MINUS
